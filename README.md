Exercícios de Lógica de Programação em Java

Este repositório contém exercícios básicos de lógica de programação em Java, desenvolvidos como parte de uma revisão e consolidação dos fundamentos da linguagem.

Os exercícios abordam conceitos como:

- entrada, processamento e saída de dados;

- estruturas condicionais;

- estruturas de repetição;

- organização básica de código.

O objetivo deste projeto é reforçar a base lógica, praticar a sintaxe da linguagem Java e manter um histórico organizado de estudos no GitHub.
